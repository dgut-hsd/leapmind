# LeapMind AI 服务 - 测试指南

## 启动服务
```bash
conda activate leapmind
cd aitutor-backend-python
python run_leapmind.py
```
然后打开 http://localhost:8001/docs

---

## 测试步骤

### 测试1：印刷体选择题识别
**对应 3.8 第一项：印刷体题目识别**

在 Swagger 中找到 `POST /api/internal/ai/ocr/process` → Try it out

Request body 填入：
```json
{
  "rawText": "在直角三角形ABC中，∠C=90°，AC=3，BC=4，则AB=？ A. 5 B. 6 C. 7 D. 8",
  "subject": "math"
}
```

**预期结果**：stem=题干，options=["5","6","7","8"]，type=single_choice，confidence>0.8

---

### 测试2：填空题识别
**对应 3.8 第五项：填空题识别**

```json
{
  "rawText": "勾股定理公式是：a² + b² = ____",
  "subject": "math"
```

### 测试3：数学公式保留
**对应 3.8 第三项：数学公式**

```json
{
  "rawText": "如图，∠A=30°，∠B=60°，则∠C=____。 A. 90° B. 80° C. 70° D. 60°",
  "subject": "math"
}
```

**预期结果**：∠、° 等符号保留正确

---

### 测试4：AI答疑
**对应 3.8 第六项：答疑 Prompt 输出**

找到 `POST /api/internal/ai/generate` → Try it out

```json
{
  "module": "explain",
  "scene": "photo_qa",
  "params": {
    "question": {
      "stem": "一个长方形的长是6cm，宽是4cm，求面积。",
      "options": ["20cm²", "24cm²", "10cm²", "30cm²"],
      "type": "single_choice",
      "subject": "math"
    }
  }
}
```

**预期结果**：返回内容包含"答案"、"解析步骤"、"涉及知识点"、"易错点提醒"

### 测试5：AI讲题（个性化）
**对应 3.8 第七项：讲题个性化输出**

```json
{
  "module": "explain",
  "scene": "explain_wrong",
  "params": {
    "question": {
      "stem": "计算 2+3×4 的值。",
      "options": ["14", "20", "24", "10"],
      "correctAnswer": "14"
    },
    "userAnswer": {"selected": "20"},
    "wrongReasonTag": "method_wrong",
    "knowledgePoints": [{"id": 1, "name": "运算优先级"}]
  }
}
```

**预期结果**：讲解内容针对"解题方法选择错误"这个原因展开，说明为什么先乘除后加减

### 测试6：SSE流式返回
**对应 3.8 第八项：SSE 流式返回**

找到 `POST /api/internal/ai/generate/stream`，填入跟测试4一样的参数。

**预期结果**：内容逐段返回（打字机效果），首段在2秒内出现

---

### 常见问题
- Execute 后看到 `Response body` 区域就是返回结果
- 如果报错 `422`，说明参数格式不对，检查 JSON 是否有拼写错误
- 如果报 `timeout`，可能是 API Key 没配好或网络问题
