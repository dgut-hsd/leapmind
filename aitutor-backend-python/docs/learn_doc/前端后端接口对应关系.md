## 📌 使用指南

### ⚡ 快速开始

1. **项目初始化**：
   ```bash
   # 克隆项目
   git clone https://github.com/your_repo/landppt.git
   cd landppt

   # 创建虚拟环境
   python -m venv venv
   source venv/bin/activate # Linux/Mac
   venv\Scripts\activate # Windows

   # 安装依赖
   pip install -r requirements.txt
   ```

2. **启动后端**：
   ```bash
   # 启动开发服务器
   uvicorn web.main:app --reload
   ```

3. **启动前端**：
   ```bash
   # 进入前端目录
   cd src/landppt/web

   # 安装依赖
   npm install

   # 启动开发服务器
   npm run dev
   ```

4. **访问应用**：
   ```
   http://localhost:3000
   ```
