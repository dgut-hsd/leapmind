## 🎯 核心技术特性

### 1. JWT Token 认证
```javascript
// Token 自动管理
const token = getToken(); // 获取 token
if (token) {
  headers['Authorization'] = `Bearer ${token}`;
}
```

### 2. 自动过期检查
```javascript
// Token 过期自动检测
const expiryTime = localStorage.getItem(TOKEN_EXPIRY_KEY);
if (expiryTime && Date.now() > parseInt(expiryTime)) {
  removeToken(); // 自动清除过期 token
  return null;
}
```

### 3. 统一错误处理
```javascript
// 自定义错误类
export class ApiError extends Error {
  constructor(message, code, data = null) {
    super(message);
    this.code = code;
    this.data = data;
  }
}
```

### 4. 响应拦截
```javascript
// 401 自动处理
if (response.status === 401 || result.code === 401) {
  removeToken(); // 清除无效 token
  throw new ApiError('未授权访问，请重新登录', 401);
}
```

## 💡 使用示例

### 完整登录流程
```jsx
import { login, ApiError } from '../services';

async function handleLogin(username, password) {
  try {
    // 1. 调用登录 API
    const result = await login(username, password);
    
    // 2. Token 和用户信息已自动保存到 localStorage
    console.log('登录成功！');
    console.log('Token:', result.token);
    console.log('用户:', result.user);
    
    // 3. 更新应用状态或刷新页面
    window.location.reload();
    
  } catch (error) {
    // 4. 处理错误
    if (error instanceof ApiError) {
      if (error.code === 401) {
        alert('用户名或密码错误');
      } else if (error.code === 400) {
        alert('请求参数错误');
      } else {
        alert('登录失败: ' + error.message);
      }
    } else {
      alert('网络错误，请检查连接');
    }
  }
}
```

### 受保护的 API 调用
```jsx
import { getUserProfile } from '../services';

async function loadUserData() {
  try {
    // Token 会自动添加到请求头
    const user = await getUserProfile();
    console.log('当前用户:', user);
  } catch (error) {
    // 如果 token 无效，会自动清除并抛出 401 错误
    console.error('获取用户信息失败:', error);
  }
}
```

## 🔒 安全特性

1. **Token 存储**: 使用 localStorage（建议生产环境使用 HTTPS）
2. **自动过期**: Token 过期自动清除
3. **401 拦截**: 无效 token 自动清除并跳转登录
4. **错误隔离**: try-catch 包裹所有 API 调用
5. **密码安全**: 前端不存储密码
