# AI教育平台前端项目 - 项目结构详解

## 📋 项目概览

这是一个 **AI虚拟教师交互系统** 的前端项目，基于以下技术栈：

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 19.1.1 | UI框架 |
| Three.js | 0.179.1 | 3D图形库 |
| VRM | @pixiv/three-vrm 3.4.2 | 虚拟人物模型格式 |
| Vite | 7.1.0 | 构建工具 |
| Tailwind CSS | 4.1.11 | 样式框架 |

**核心功能：**
- 🎓 学生可通过项目/课程与AI虚拟教师交互
- 🤖 虚拟教师能进行头部动作、表情控制、语音合成
- 📊 支持幻灯片讲课、语音互动、表情同步
- 🎬 实时口型同步（LipSync）

## 🎯 关键目录说明

### 1. **pages/** - 页面级组件（应用入口）

这些是顶级页面，直接由 `App.jsx` 路由到：

```jsx
// App.jsx 的路由逻辑
isChecking ? <Loading /> : !isAuthed ? <LoginPage /> : projectId ? <LecturePage /> : <ProjectListPage />
```

| 页面 | 职责 | 关键状态 |
|------|------|---------|
| `LoginPage` | 用户登录 | 认证token |
| `ProjectListPage` | 显示课程项目列表 | projectId |
| `LecturePage` | ⭐ 核心教学界面 | 幻灯片+虚拟教师 |

### 2. **components/** - UI组件层（展示层）

- **chat/** - 聊天相关UI
- **lecture/** - 讲课相关UI（幻灯片显示）
- **teacher/** - 教师虚拟人物相关UI
- **character/** - 3D角色相关UI组件

### 3. **features/** - 业务逻辑层（最重要！）

这是项目的"大脑"，包含所有业务逻辑：

#### A. **vrmViewer/** - 3D图形渲染系统
```
viewer.js 
  ├─ Three.js场景管理
  ├─ 相机、光照、渲染器
  └─ 动画循环

model.js
  ├─ VRM模型加载
  ├─ 动画混合器
  └─ 表情控制器
```

#### B. **emoteController/** - 表情和动作系统
```
emoteController.js (主入口)
  ├─ ExpressionController (表情)
  │   └─ 喜悦、悲伤、愤怒等
  │
  └─ HeadMotionController (头部动作)
      ├─ 点头、摇头、歪头等
      └─ 关键词触发
```

#### C. **lipSync/** - 口型同步系统
```
lipSync.js
  ├─ 音频分析（FFT）
  ├─ 音量包络检测
  └─ 口型权重计算
```

#### D. **chat/** - PPT讲课系统
```
pptState.js (状态管理)
pptController.js (播放控制)
pptPlayback.js (播放逻辑)
pptVoiceControl.js (语音识别)
pptApi.js (后端API)
```

### 4. **lib/** - 第三方库和扩展

- **VRMAnimation/** - VRM格式的动画加载和播放
- **VRMLookAtSmootherLoaderPlugin/** - VRM注视目标平滑插件

## 📊 数据流向

### 流程1：用户提问时
```
用户输入
  ↓
ChatInput.jsx
  ↓
handleSendMessage (TeacherPanel)
  ↓
pptApi.askQuestion() → 后端OpenAI处理
  ↓
pptApi.synthesizeSpeech() → 火山引擎TTS
  ↓
model.speak(audioBuffer, screenplay)
  ├─ LipSync分析音频
  ├─ EmoteController控制表情和头部动作
  └─ 虚拟人物说话并做动作
```

### 流程2：PPT讲课时
```
SlideViewer加载PPT
  ↓
pptController.startPlayback()
  ↓
播放PPT音频片段
  ├─ 同时触发语音识别
  └─ 更新字幕和虚拟人物动作
  ↓
用户语音打断
  ↓
handleVoiceInterruption()
  ├─ 暂停PPT播放
  ├─ 调用AI回答
  ├─ TTS合成和播放
  └─ 恢复PPT播放
```
