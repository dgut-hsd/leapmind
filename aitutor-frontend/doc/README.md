## 🎓 关键概念导航

### 最重要的三个概念

#### 1. 虚拟人物的三层表现

```
Layer 1: 头部动作 (HeadMotionController)
         ├─ 点头、摇头、歪头、抬头
         └─ 通过关键词或情绪触发

Layer 2: 表情 (ExpressionController)
         ├─ 开心、悲伤、愤怒、惊讶等
         └─ 自动跟随头部动作

Layer 3: 口型 (LipSync)
         ├─ 根据音频实时分析
         └─ 自动调整嘴型
```

#### 2. 数据流向

```
用户输入
  ↓
后端API调用
  ↓
语音合成
  ↓
虚拟人物播放
  ├─ 头部动作
  ├─ 表情
  └─ 口型同步
  ↓
渲染到屏幕
```

#### 3. 状态管理

```
React State (在组件中)
  ├─ 聊天消息
  └─ UI显示

全局State (pptState)
  ├─ 播放状态
  ├─ 当前片段
  └─ 中断标记

3D Viewer State
  ├─ 虚拟人物位置
  ├─ 骨骼旋转
  └─ 表情权重
```
