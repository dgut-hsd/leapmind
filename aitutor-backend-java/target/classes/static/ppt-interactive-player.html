📖 使用说明

1. 权限设置：首次使用需要允许浏览器访问麦克风权限

2. 语音识别：点击"🎤 测试语音识别"按钮测试功能是否正常

3. 播放控制：加载音频后点击"开始播放"，系统会播放整个页面的音频并启动语音监听

4. 实时监听：系统会在播放过程中持续监听您的语音

5. 声音激活打断：在播放过程中开口说话会立即暂停讲课

6. 智能识别：系统会过滤低置信度的识别结果，确保准确响应

7. 智能恢复：AI回答完成后，系统会根据设置从句子开头或打断位置继续播放讲课

8. 恢复模式：可选择"从句子开头恢复"（更自然）或"从打断位置恢复"（更精确）

9. 浏览器支持：推荐使用Chrome、Edge浏览器，确保网络连接稳定

🔧 故障排除

网络错误：检查网络连接，确保可以访问Google语音服务

权限被拒绝：点击地址栏左侧的锁图标，允许麦克风访问

无法识别语音：确保麦克风工作正常，说话清晰

识别不准确：在安静环境中使用，避免背景噪音

🧩 批量音频合成

将 PPT 的结构化数据（含标题与各页内容点）粘贴为 JSON，点击开始批量合成。

智能润色功能：系统会根据页面类型（如目录、标题页、内容页等）自动调整讲解风格，并严格控制润色后的字数不超过原文+200字，确保内容精炼适度。
